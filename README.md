# Cinematheque_mavie
Application Web orientée donnée et API développée dans le cadre du projet OpsCI (L3 Informatique Sorbonne Université)

##  Présentation du projet

Dans le cadre du projet d'OpsCI proposé au semestre 6 de la 3ᵉ année de licence monodisciplinaire informatique, nous avons conçu et déployé une application web orientée données et API.

## Architecture 

Notre application à une architecture à 3 couches, client-serveur.

##  Technologies utilisées

- **Backend** : FastAPI  
- **Frontend** : React (avec Vite)  
- **Base de données** : PostgreSQL  
- **Conteneurisation** : Docker  
- **CI/CD** : GitLab CI/CD  

## Données

Les données utilisées dans ce projet proviennent d’un dataset disponible sur Kaggle.

## Compétences acquises

Ce projet nous a permis de :

- Mettre en place une API avec FastAPI  
- Développer une interface moderne avec React  
- Gérer une base de données PostgreSQL  
- Automatiser les tests et le déploiement avec GitLab CI/CD  
- Conteneuriser une application avec Docker  
- Découvrir des outils comme Kubernetes et Kafka
